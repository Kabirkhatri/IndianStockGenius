# Indian Stock Genius

Indian Stock Genius is an interactive stock market analysis and prediction platform focused on the Indian equity market. Built primarily with Python and Streamlit, this application combines fundamental analysis, technical chart pattern recognition, price prediction, and sentiment analysis to aid retail investors, analysts, and enthusiasts in making informed decisions.

---

## Features

- **Streamlit Web Interface:** User-friendly, interactive frontend for analysis and visualization.
- **Data Fetching:** Automated retrieval of stock data for Indian market equities.
- **Technical Analysis:** Detection of candlestick patterns and other technical indicators.
- **Fundamental Analysis:** Parsing and reporting of key financial metrics of companies.
- **Price Prediction:** Predictive modeling to forecast stock price trends.
- **Sentiment Analysis:** Incorporation of market and news sentiment to assess psychological market factors.
- **Report Generation:** Generate detailed analytical reports combining multiple insights.

---

## Installation

### Prerequisites

- Python 3.8 or higher
- pip package manager

### Setup

```
git clone https://github.com/Kabirkhatri/IndianStockGenius.git

cd IndianStockGenius

python -m venv venv

source venv/bin/activate    # Windows: venv\Scripts\activate

pip install -r requirements.txt
```

---

## Usage

Run the Streamlit app locally:

```
streamlit run app.py
```

Access the app through the URL provided in your terminal (usually `http://localhost:8501`).

## Code Structure

| File                 | Description                                         |
|----------------------|-----------------------------------------------------|
| `app.py`             | Main Streamlit application entry point              |
| `chart_patterns.py`  | Technical pattern recognition logic                  |
| `data_fetcher.py`    | Module to fetch and preprocess stock market data    |
| `fundamental_analysis.py` | Extracts and analyzes company fundamentals      |
| `price_prediction.py`| Implements forecasting models                        |
| `report_generator.py`| Generates comprehensive analysis reports             |
| `sentiment_analysis.py`| Analyzes news and market sentiment                  |
| `technical_analysis.py`| Additional technical indicators                      |
| `utils.py`           | Helper functions                                     |
